# P2P SAP Project (Procurement Management System)

## Overview
A Flask-based web application to manage procurement workflow:
PR → PO → GR → Invoice.

## Features
- Create Purchase Requests (PR)
- Create Purchase Orders (PO)
- View records on dashboard
- SQLite-based storage

## Tech Stack
- Python (Flask)
- SQLite
- HTML/CSS

## How to Run
pip install flask  
python database.py  
python app.py
